# Twixbus

Startup que facilita el pago del transporte publico en la ciudades

- _Paga escaneando código QR instalado en el bus_

## Tech Stack

**Client:** React-Native, Expo, Redux

**Server:** Firebase

## Características

- Offline mode
- Inicio de sesión sin crear cuentas
- Escanea código QR
